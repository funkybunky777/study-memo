## PSQL

* 사용자 조회

	* ```select * from pg_shadow;```


* 비밀번호 변경

	* ```alter user dodotdo with password 'ektenekt25';```

* Create User
	* ```create user username [[with] options [...]] where option can be;```
	* ```create user test2 password 'test2';```
